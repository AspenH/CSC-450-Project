Project FandRec

Requirements:
  Python v3.5 or higher
   pip install these:
    wsaccel, ujson, autobahn, twisted, opencv-python, opencv-contrib-python

1. start the server using:
    python slate.py

2. start the client using:
    python camera_client.py

3. connect to the webpage using:
    127.0.0.1:8090
